## gibbon-mail frontend

```
$ yarn install
```

Before start `frontend` you need to start the [`../backend/`](../backend/) service.

```
$ export REACT_APP_API_URL=http://127.0.0.1:5000
$ yarn run start
```

Open your browser on http://127.0.0.1.3000
